# EFL Clone 🌐

> A pixel-faithful recreation of [EFL Website](https://endlessfrontierlabs.com/) — built as a Software Engineering Internship task at **NYU's Endless Frontier Labs**.

🔗 **Live Demo:** [https://efl-clone.com](https://efl-clone.free.nf/)

---

## 📌 About

This project is a full front-to-back clone of the Endless Frontier Labs (EFL) website. It replicates the original site's design, layout, and functionality — built entirely from scratch to demonstrate real-world web development skills including responsive design, server-side rendering, and database integration.

## ✨ Features

- Faithful recreation of the EFL website UI and layout
- Fully responsive across desktop and mobile
- Dynamic content rendering with PHP
- MySQL-backed database structure
- Clean, maintainable codebase

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| **HTML5 & CSS3** | Page structure and styling |
| **JavaScript** | Interactivity and dynamic content |
| **Bootstrap** | Responsive grid and UI components |
| **PHP** | Server-side logic and templating |
| **WordPress** | CMS integration |
| **MySQL** | Database via `database_setup.sql` |

## 🚀 Getting Started

```bash
# 1. Clone the repo
git clone https://github.com/Vikram739/efl-clone.git

# 2. Set up a local PHP server (XAMPP / WAMP)
# 3. Import database_setup.sql into MySQL
# 4. Open index.php in your browser
```
